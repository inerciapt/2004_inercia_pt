compo machine:
p4 512ram or better
radeon9600 or better
stereo sound

demo compo:
no writing to hard drive, 12min limit (or will be stopped), 20mb limit (20.971.520 bytes), smaller sizes preferred, must exit on 'esc' or alt+f4. compo machine will have dx9 compatible video card, if you have other hardware requirements bring your own equipment to show. all binaries must be delivered.

64k intro:
same rules as demo compo with 64kb (65.536 bytes) size limit plus, size limit is applicable to zip or exe (which ever is smaller).

4k intro:
same rules as demo compo with 4k (4096 bytes) size limit plus, size limit is applicable to zip or exe (which ever is smaller).

pro music:
5 minutes maximum duration. if the limit is exceeded, it will be faded out. mp3 or ogg at VBR or 160kbps minimum.

tracked music:
5 minutes maximum duration. if the limit is exceeded, it will be faded out. max file size of 1mb (1.048.576 bytes). mod, s3m, xm, it (will be played in ModPlug Player) and rns WITHOUT plugins (will be played on Renoise).

freestyle graphics:
maximum resolution at 800x600x24bit. png, jpeg or gif are accepted. must state tools/sources used on info file and must include at least 5 steps of creational process (for pre-selection purposes only, these will not be in the party release files). these images must be named *_(01-05).xxx and be included in a *.zip file. images will be shown using latest ACDSee.

fast gfx:
theme will be given during the party. participants will have a time limit to create their image. accepted formats are the same as in the freestyle graphics competition.

wild compo:
size limit of 100mb (104.857.600bytes) and no longer than 15 minutes. DivX or XVid (open-source) codec’s preferred. please try NOT to use 'exotic' codecs. submission will be via CD with info and video file only (and codecs if different than preferred).

gamedev:
10mb size limit. no theme or platform limitation. this compo deadline will be on the first day of the party, so bring your game finished. we will spread the game through the network and let you demonstrate/present it on-screen for 5minutes using our compo machine, your own hardware or a video recording of it (in case it's a very exotic hardware not suitable for realtime big screen demonstration).

further info/discussion on compo rules can be read/posted @ scene.org forums
